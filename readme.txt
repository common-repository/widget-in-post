Insert any widget into a post or page.
Use All Widget parameters with a GUI.

== Description == 

The plugin adds virtual widget areas to your widget setting screen.
Drag and drop a widget or widgets to the area, and use the shortcode to show it on the page.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `sc-google-maps` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
